Abstract

This paper presents an online tool for obesity level prediction using a 1D Convolutional Neural Network (CNN) model. The tool operates entirely within the user's web browser, ensuring data privacy as no personal information is transmitted to external servers. The model utilizes 16 anthropometric, dietary, and lifestyle features to classify individuals into seven obesity categories based on the WHO classification system. The implementation demonstrates the feasibility of deploying deep learning models for health assessment in privacy-sensitive contexts.

Keywords: Deep learning layers, Obesity, Web-based application, 1D-CNN

Methodology

The 1D CNN model was trained on a dataset of 2,111 individuals with 16 features including demographic, anthropometric, dietary, and lifestyle variables. The model architecture consists of three convolutional layers with ReLU activation, followed by max-pooling layers and two fully connected layers. The model achieves an accuracy of 95.02% on the test set. The tool is implemented using ONNX runtime for browser-based inference.

Model Specifications:
  • Architecture: 1D CNN with 3 convolutional layers
  • Input: 16 normalized features
  • Output: 7 obesity classes
  • Accuracy: 95.73% (validation set)

Technical Implementation

The prediction tool was implemented using MATLAB for model development, the trained network was converted to ONNX format, and run using ONNX Runtime Web. The 1D CNN architecture processes 16 input features through three convolution blocks with batch normalization and dropout adjustment.
